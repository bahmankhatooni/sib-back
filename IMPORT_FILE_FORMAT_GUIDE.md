# راهنمای قالب فایل Import

## خطای رایج: "عناوین فیلدها یافت نشد"

این خطا زمانی رخ می‌دهد که شیت Excel شما فرمت استاندارد را ندارد.

---

## ✅ فرمت صحیح شیت

هر شیت در فایل Excel باید **حداقل 6 ردیف** داشته باشد:

```
ردیف 1 (A1): هدف X : عنوان هدف
ردیف 2 (A2): برنامه Y : عنوان برنامه
ردیف 3 (A3): اقدام Z : عنوان اقدام
ردیف 4 (A4): فعالیت W : عنوان فعالیت
ردیف 5 (A5): ردیف
ردیف 6 (A6, B6, C6, ...): ردیف | فیلد 1 | فیلد 2 | فیلد 3 | ...
ردیف 7+ (A7+, B7+, ...): 1 | مقدار 1 | مقدار 2 | مقدار 3 | ...
```

---

## 📋 توضیحات هر ردیف

### ردیف 1: هدف (الزامی)
```
فرمت: هدف [کد] : [عنوان]
مثال: هدف 5 : به ثمر رساندن تمامی پرونده ها
```

### ردیف 2: برنامه (الزامی)
```
فرمت: برنامه [کد] : [عنوان]
مثال: برنامه 4 : انجام اقدامات حقوق عامه ای
```

### ردیف 3: اقدام (الزامی)
```
فرمت: اقدام [کد] : [عنوان]
مثال: اقدام 30201 : نظارت بر امور محیط زیست
```

### ردیف 4: فعالیت (اختیاری)
```
فرمت: فعالیت [کد] : [عنوان]
مثال: فعالیت 1 : پیگیری و نظارت بر عملکرد
```

### ردیف 5: عنوان "ردیف" (الزامی)
```
سلول A5 باید شامل کلمه "ردیف" باشد
```

### ردیف 6: عناوین فیلدها (اختیاری اما توصیه می‌شود)
```
A6: ردیف
B6: نام فیلد اول
C6: نام فیلد دوم
D6: نام فیلد سوم
...
```

⚠️ **نکته مهم:** اگر شیت شما فقط metadata دارد (ردیف‌های 1-5) و **هیچ فیلدی ندارد**، کاربرگ بدون فیلد ایجاد می‌شود.

### ردیف 7 به بعد: داده‌ها (اختیاری)
```
A7: 1
B7: مقدار فیلد اول در ردیف 1
C7: مقدار فیلد دوم در ردیف 1
...

A8: 2
B8: مقدار فیلد اول در ردیف 2
C8: مقدار فیلد دوم در ردیف 2
...
```

## ❌ حالت‌های خطا

### خطای 1: شیت کمتر از 6 ردیف دارد
```
❌ شیت فقط 3 ردیف دارد
✅ شیت حداقل 6 ردیف داشته باشد (حتی اگر خالی باشند)
```

### خطای 2: ردیف 6 کاملاً خالی است
```
❌ ردیف 6: [خالی] [خالی] [خالی]
✅ ردیف 6: ردیف | نام | تاریخ | نتیجه

یا

✅ شیت را بدون ردیف 6 (فقط metadata) import کنید
```

### خطای 3: فرمت ردیف‌های 1-4 نادرست است
```
❌ ردیف 1: "5 : به ثمر رساندن"
✅ ردیف 1: "هدف 5 : به ثمر رساندن"

❌ ردیف 2: "برنامه ۴"
✅ ردیف 2: "برنامه 4 : انجام اقدامات"
```

## 🎯 مثال فایل صحیح

### مثال 1: شیت با فیلدها و داده‌ها

**شیت: "80302-1"**

| A | B | C | D | E |
|---|---|---|---|---|
| هدف 8 : اصلاح قوانین | | | | |
| برنامه 3 : بررسی قوانین موجود | | | | |
| اقدام 80302 : تحلیل قوانین مرتبط | | | | |
| فعالیت 1 : جمع‌آوری اطلاعات | | | | |
| ردیف | | | | |
| ردیف | نام پرونده | تاریخ | نتیجه | توضیحات |
| 1 | پرونده الف | 1403/03/01 | بررسی شد | کامل |
| 2 | پرونده ب | 1403/03/05 | در انتظار | ناقص |
| 3 | پرونده ج | 1403/03/10 | نهایی شد | تایید |

**نتیجه:**
- ✅ کاربرگ با کد "80302-1" ایجاد می‌شود
- ✅ 4 فیلد: نام پرونده، تاریخ، نتیجه، توضیحات
- ✅ 3 ردیف داده

### مثال 2: شیت فقط با metadata (بدون فیلد)

**شیت: "80401-2"**

| A |
|---|
| هدف 8 : اصلاح قوانین |
| برنامه 4 : ارائه پیشنهادات |
| اقدام 80401 : تدوین پیشنهادات |
| فعالیت 2 : بررسی نهایی |
| ردیف |
| [خالی] |

**نتیجه:**
- ✅ کاربرگ با کد "80401-2" ایجاد می‌شود
- ✅ **بدون فیلد** (چون ردیف 6 خالی است)
- ✅ بدون داده

## 🔧 راه‌حل‌های خطا

### راه‌حل 1: اضافه کردن ردیف 6 با عناوین
اگر شیت شما داده دارد، حتماً ردیف 6 را با عناوین فیلدها پر کنید:

```
قبل:
ردیف 5: ردیف
ردیف 6: [خالی]
ردیف 7: 1 | مقدار 1 | مقدار 2

بعد:
ردیف 5: ردیف
ردیف 6: ردیف | فیلد 1 | فیلد 2
ردیف 7: 1 | مقدار 1 | مقدار 2
```

### راه‌حل 2: حذف شیت‌های خالی یا ناقص
اگر شیتی فقط metadata دارد و نمی‌خواهید کاربرگ خالی ایجاد شود، آن شیت را حذف کنید.

### راه‌حل 3: بررسی تمام شیت‌ها قبل از import
قبل از import، تمام شیت‌ها را باز کنید و مطمئن شوید:
- ✅ ردیف‌های 1-5 فرمت صحیح دارند
- ✅ ردیف 6 یا دارای عناوین است یا کاملاً خالی
- ✅ ردیف‌های 7+ داده دارند یا وجود ندارند

---

## 📊 بررسی سریع فایل

برای بررسی فایل خود قبل از import:

1. **باز کردن فایل Excel**
2. **رفتن به هر شیت**
3. **چک کردن:**
   - آیا ردیف 1 شامل "هدف" است؟
   - آیا ردیف 2 شامل "برنامه" است؟
   - آیا ردیف 3 شامل "اقدام" است؟
   - آیا ردیف 4 شامل "فعالیت" است؟
   - آیا ردیف 5 شامل "ردیف" است؟
   - آیا ردیف 6 دارای عناوین یا خالی است؟

---

## ⚠️ نکات مهم

1. **کد هدف و برنامه باید در سیستم موجود باشد**
   - اگر خطای "هدف یافت نشد" دریافت کردید، ابتدا هدف را در سیستم ثبت کنید

2. **اقدام و فعالیت اختیاری هستند**
   - اگر اقدام یافت نشد، کاربرگ بدون اقدام ایجاد می‌شود
   - اگر فعالیت یافت نشد، کاربرگ بدون فعالیت ایجاد می‌شود

3. **نام شیت = کد کاربرگ**
   - نام هر شیت به عنوان کد کاربرگ استفاده می‌شود
   - مطمئن شوید نام شیت‌ها یکتا هستند

4. **شیت‌های بدون فیلد**
   - اگر شیت فقط metadata دارد، کاربرگ بدون فیلد ایجاد می‌شود
   - می‌توانید بعداً فیلدها را از طریق "مدیریت فیلدها" اضافه کنید

## 🎓 مثال کامل برای 500 کاربرگ

فایل Excel خود را اینگونه آماده کنید:

```
📄 کاربرگ‌های-1403.xlsx
│
├── 📑 شیت 1: "30201-1-1403-01"
│   ├── ردیف 1-5: metadata
│   ├── ردیف 6: عناوین
│   └── ردیف 7+: داده‌ها
│
├── 📑 شیت 2: "30201-2-1403-01"
│   ├── ردیف 1-5: metadata
│   ├── ردیف 6: عناوین
│   └── ردیف 7+: داده‌ها
│
... (498 شیت دیگر)
```

**نتیجه:**
```
✅ 500 کاربرگ با موفقیت ایجاد شد
```

یا در صورت خطا:
```
✅ 485 کاربرگ با موفقیت ایجاد شد

⚠️ خطاها:
- شیت '80302-2': عناوین فیلدها یافت نشد
- شیت '80401-2': عناوین فیلدها یافت نشد
...
```
