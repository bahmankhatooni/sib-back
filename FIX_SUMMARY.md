## 🔍 بررسی نتیجه در دیتابیس

```sql
-- بررسی آخرین کاربرگ
SELECT 
    f.id,
    f.code,
    f.task_id,
    t.code as task_code,
    (SELECT COUNT(*) FROM form_fields WHERE form_id = f.id) as field_count,
    (SELECT COUNT(*) FROM form_field_values WHERE form_id = f.id) as value_count
FROM forms f
LEFT JOIN tasks t ON f.task_id = t.id
ORDER BY f.created_at DESC
LIMIT 1;
```

**نتیجه مورد انتظار:**
```
| id | code | task_id | task_code | field_count | value_count |
|----|------|---------|-----------|-------------|-------------|
| 10 | test-001 | 5 | 30201 | 2 | 2 |
```
